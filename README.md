## Features
- [x] 基础页面
  - [x] 首页信息流
  - [x] 排行榜(包括 AI 排行榜)
  - [x] 作品页面
  - [x] 作者信息页面
  - [x] 设置页面
- [x] 搜索功能
- [x] 以图搜图
- [x] 动图播放
- [x] 动图下载(ZIP/GIF/WebM)
- [x] 使用 localforage 存储缓存
- [x] 历史记录
- [x] 多端样式适配
- [x] PWA 支持
- [x] R18 与 AI 作品开关
- [x] 信息流布局选择
- [x] 多图床选择
- [x] 多 API 实例选择
- [x] 首页特辑/推荐/发现/新作
- [x] 搜索自动补全
